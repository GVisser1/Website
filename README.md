# Website

Personal Website

## Available Scripts

- `pnpm install` to install dependencies
- `pnpm dev` to start dev environment, with live-reload. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
- `pnpm build` to build the app for production
- `pnpm test` to run unit tests
- `pnpm test:e2e` to run e2e tests
